# Review of Data Review Tool

-  Edit selection tool at the top allows highlighting a period of time with a yellow box to
- After highlighting an area, the edit tab opens up and shows all the actions that can be applied to the data
  - Edit types that edit/correct/create data + ones that apply metadata

## Edit Categories - Edits 
- Clean up inacurate data 
  - Data spikes, etc

## Edit Categories - Corrections
- Truth checks that adjustments can be made to 

## Edit Categories - Create/Estimate 
- Single Point, Gap Fill, Override 
- Estimate when there is a rule that data has to be there

## Aquarius Time Series Action Priorities 
- Ensure that calculations are applied in a logical order
- Pre
  - Edits, etc that should be done early  
- Normal
  - stage/WL calculations done after pre and before post
- Post 
  - inputting a peak stage, actions that are done later
- Thresholds are applied last
- Correction stack can be viewed in the edit tab
